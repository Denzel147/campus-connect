# CampusConnect Presentation Files

This folder contains presentation materials for the CampusConnect project.

## Files

1. **PRESENTATION.html** - Interactive HTML slideshow (Ready to use!)
2. **PRESENTATION_OUTLINE.md** - Detailed slide-by-slide outline for PowerPoint
3. **PRESENTATION_README.md** - This file (instructions)

## Quick Start

### Option 1: Use HTML Presentation (Recommended)

1. **Open `PRESENTATION.html`** in any modern web browser
2. **Fullscreen Mode:** Press `F11` for presentation mode
3. **Navigation:**
   - `‚Üí` or `Space` - Next slide
   - `‚Üê` - Previous slide
   - `ESC` - Slide overview
   - `F` - Fullscreen toggle
4. **Export to PDF:**
   - Press `Ctrl+P` (or Cmd+P on Mac)
   - Select "Save as PDF"
   - Choose "Save background graphics" for best quality

### Option 2: Create PowerPoint from Outline

1. **Open PowerPoint** and create a new presentation
2. **Open `PRESENTATION_OUTLINE.md`** as a reference
3. **For each slide in the outline:**
   - Create a new slide
   - Copy the title and content
   - Format consistently

**Pro Tips:**
- Use the color scheme: Primary Blue (#2563eb), Success Green (#059669)
- Include screenshots from the actual application
- Add the architecture diagrams from `docs/ARCHITECTURE_DIAGRAMS.md`
- Use SmartArt for feature comparisons

## Customization

### To Customize the HTML Presentation:

1. Edit `PRESENTATION.html` in a text editor
2. Modify content within `<section>` tags
3. Adjust colors in the `<style>` section
4. Save and refresh browser

### To Customize PowerPoint:

1. Follow the outline in `PRESENTATION_OUTLINE.md`
2. Add your own branding/colors
3. Include screenshots of your actual application
4. Add team photos or additional content

## Tips for Presentation

- **Timing:** Each slide ~1-2 minutes = ~18-36 minute presentation
- **Demo:** Prepare live demo on Slide 16
- **Q&A:** Leave time at the end (Slide 18)
- **Backup:** Have both HTML and PowerPoint versions ready

## System Requirements

**HTML Presentation:**
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection (for CDN resources) - OR download reveal.js locally

**PowerPoint:**
- Microsoft PowerPoint (2016 or later recommended)
- Or Google Slides, LibreOffice Impress for alternatives
